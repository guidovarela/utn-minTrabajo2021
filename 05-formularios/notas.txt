
Formularios

Estrucutura de HTML que le permite al usuario ingresar informacion -> interaccion

<form action="" method="" id="">
    action -> definimos a donde mandamos los datos
    method -> elegir el metodo de envio -> POST / GET

    <input type="" name="">
        Campos de linea simple / seleccionables
        type -> tipo de campo (text, email, password, checkbox, radio, color, date, number, submit, button, etc...)
        name -> nombre del campo, que es una variable. Sin este attributo, no se envia el dato

    <select name="">
        <option value="1"> Nombre de la opcion </option>
        <option value="" > </option>
    </select>

    <textarea name="">
        Campo de texto multilinea
    </textarea>

    <label></label> -> vincular el titulo del campo, con el campo. Es una ayuda de accesibilidad -> clic en el texto, foco en el campo

    /* Atributos HTML5*/
    - required -> Validacion de los datos, deteccion de errores (campo vacio, texto que no coincida con el type, etc)
    - pattern="" -> Aplicar una expresion regular para mejorar la validacion de datos
    - placeholder="" -> agrega un texto de ayuda en el campo
    - autofocus -> Foco automatico
    
</form>
